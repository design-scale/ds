# Design System
Construa e escale seu Design System com facilidade. 
Cuidando de todos os aspectos, desde uma documentação 
poderosa até uma gestão eficiente do sistema de design, 
esse ecossistema poderoso acompanha o crescimento do seu design system. 
Todos os recursos essenciais para estabelecer o sistema de design estão 
disponíveis gratuitamente, e para sempre. Otimize sua experiência agora 
mesmo!





### PARA OS TIMES

- Consistência e coerência visual;
- Menos decisões de design;
- Maior direção criativa;
- Fluxo de trabalho eficiente;
- Melhor comunicação entre designers e desenvolvedores.

### PARA OS USUÁRIOS
- Experiência de uso unificada;
- Saber quais funcionalidades esperar dos componentes;
- Saber onde encontrar determinadas funcionalidades na página;
- Saber como usar uma funcionalidade para atingir seu objetivo;
- Não se esforçar para entender o significado de elementos desconhecidos;
- Aumentar a sensação de domínio das ferramentas e a habilidade de concluir suas atividades melhorando sua satisfação com a experiência.

### PARA O NEGÓCIO

- Melhores estimativas para desenvolvimento;
- Menos horas escrevendo códigos e fazendo ajustes de handoff;
- Protótipos de alta fidelidade finalizados em menor tempo;
- Designers dedicando mais tempo validando as soluções com usuários;
- Produtos lançados mais rápido e com mais qualidade;
- Times mais produtivos;
- Usuários mais satisfeitos;
